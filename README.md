### 🚀 Areas of Expertise

- 🚁 **Robotics & Autonomous Systems (GNC)**: Developing guidance, navigation, and control systems. I build path planning and autonomous flight scripts (such as my control interface in [Tello](https://github.com/gonesrividya13/Tello) and UAV GNC models in [UAV_GNC](https://github.com/gonesrividya13/UAV_GNC)).
- 🧠 **Machine Learning & AI**: Training reinforcement learning agents (like [Pacman-RL](https://github.com/gonesrividya13/Pacman-RL)) and developing computer vision models (such as [Age-Detection](https://github.com/gonesrividya13/Age-Detection) and [Pose-Controlled-Tello](https://github.com/gonesrividya13/Pose-Controlled-Tello)).
- 💻 **Systems Programming**: Designing lightweight, high-performance, multithreaded systems in C/C++ (like [File-Search-Using-C](https://github.com/gonesrividya13/File-Search-Using-C) and my custom [Task-queue](https://github.com/gonesrividya13/Task-queue)).
